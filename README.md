# Crony Navigation

A forensic interface for mapping the **Epsteinia** nexus and the forces of social division.

## Methodology
- **Navigator Engine**: A reusable class-based engine for rendering dynamic, force-directed graphs.
- **Manifest Architecture**: Decoupling data (Manifests) from logic (Engine) for rapid iteration and scaling.
- **Visual Language**: Weighted edges and iconic nodes to represent the "gravity" of influence.
